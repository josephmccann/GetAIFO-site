## PROJECT OVERVIEW

You are building a production-ready marketing website for **AI.FO** (domain: getaifo.com).

**Goal:** A single-page, high-conversion landing page that matches the AI.FO pitch deck style — dark, minimal, CFO-grade. Prioritize credibility and clarity: "decision-ready judgment," not "dashboards."

---

## TECH STACK

- **Next.js** (App Router) + **TypeScript** + **Tailwind CSS**
- Airtable API for waitlist form submissions via a **Next.js API route** (`/api/waitlist`) — token is never exposed client-side
- No other backend required
- Must run with `npm run dev` and deploy cleanly on Replit

## BRAND & DESIGN SYSTEM

### Typefaces
- **Primary display**: [Bebas Neue](https://fonts.google.com/specimen/Bebas+Neue) — for large hero headlines and section titles
- **Body / UI**: [DM Sans](https://fonts.google.com/specimen/DM+Sans) — clean, modern, readable
- Import both from Google Fonts in `layout.tsx`

### Colors (CSS variables in globals.css)
```css
:root {
  --bg: #000000;
  --text-primary: #FFFFFF;
  --text-secondary: rgba(255, 255, 255, 0.72);
  --text-muted: rgba(255, 255, 255, 0.45);
  --accent: #CC6600;
  --accent-hover: #E07000;
  --border: rgba(255, 255, 255, 0.10);
  --card-bg: rgba(255, 255, 255, 0.04);
  --orange-panel: #CC6600;
}
```

### Tailwind Config
Extend Tailwind with brand colors:
```ts
extend: {
  colors: {
    accent: '#CC6600',
    'accent-hover': '#E07000',
  },
  fontFamily: {
    display: ['Bebas Neue', 'sans-serif'],
    body: ['DM Sans', 'sans-serif'],
  }
}
```

### Design Principles
- Black background throughout, lots of negative space
- Large bold display type for key phrases; tight scannable paragraphs in DM Sans
- Thin orange horizontal rule under the AI.FO wordmark in hero
- Buttons:
  - **Primary**: orange fill (`bg-accent`), black text, hover darkens
  - **Secondary**: transparent, orange border, white text, hover fills orange
- Section dividers: `border-t border-white/10`
- Cards: `bg-white/[0.04] border border-white/10 rounded-lg`
- Orange panels (used in Technical section): `bg-accent rounded-lg p-6`
- Subtle entrance animations using Tailwind `animate-` or simple CSS keyframes on scroll (use `IntersectionObserver` for reveal — keep it lightweight)

### 6. PRODUCT VISION — AUGMENTED CFO JUDGMENT (`ProductVision.tsx`)

**Section ID**: `#vision`

- Section label: "THE VISION"
- Headline: "Augmented CFO judgment. Continuous. Connected. Accountable."
- Subhead: "An AI-powered CFO layer that sits above your systems of record — ingesting accounting, banking, payroll, and operational data to produce the insight your organization actually needs."

**Feature list** (left column, 3 items with orange accent dots):
- Narrative insights leaders can actually use — not more charts
- Forward-looking risk signals before they become problems
- Decision framing tied to real choices: hiring, spend, runway, pricing
- Learns and evolves alongside the organization over time

**Right column — "CFO Brief" component**: A dark card styled like a CFO memo output, with three tabbed or stacked examples:

**Example 1 — Runway Early Warning:**
> "Cash runway has compressed from 14 months to 9 months over the past two quarters, driven primarily by a 31% increase in payroll and a slowdown in receivables collection. At the current burn rate, the organization reaches a critical threshold in Q3. Immediate action items: accelerate Q2 collections, pause two open headcount requisitions pending revenue milestone confirmation."

**Example 2 — Spend Variance Narrative:**
> "Operating expenses exceeded budget by 18% in March, concentrated in three categories: software subscriptions (+$12K), contractor spend (+$28K), and travel (+$9K). The contractor overrun is tied to the accelerated product timeline. This is a one-time variance, but if the timeline extends, a reforecast is warranted before the board meeting."

**Example 3 — Hiring Tradeoff Framing:**
> "Adding a senior engineer at $180K fully loaded increases monthly burn by $15K and compresses runway by approximately 6 weeks. At current revenue growth of 8% MoM, the breakeven on this hire is approximately 4.5 months post-ramp. Recommend proceeding if Q2 revenue lands within 10% of plan; defer if it misses by more than 15%."

Style the CFO Brief card with: dark background, subtle orange left border, monospace-ish font for the output text, small "AI.FO Analysis" label in the top corner.

**Closing line**: "A collaborative intelligence model that feels like a trusted CFO — not a dashboard, and not an automation fantasy."

### Scroll Reveal Animation
Use a lightweight `IntersectionObserver` hook (`useInView`) to add `opacity-0 translate-y-4` → `opacity-100 translate-y-0` transitions on section entry. Apply `transition-all duration-700` via Tailwind. Keep it subtle — this is a CFO product, not a SaaS marketing site.

### Accessibility
- Proper semantic heading hierarchy (h1 → h2 → h3)
- All buttons have accessible labels
- Form inputs have associated labels
- Color contrast meets WCAG AA
- Focus states visible (orange outline)

### Responsive Breakpoints
- Mobile-first
- `sm`: 640px, `md`: 768px, `lg`: 1024px, `xl`: 1280px
- All two-column layouts stack to single column on mobile
- Nav collapses to hamburger on mobile
- Hero visual panel hides on mobile (text only)
